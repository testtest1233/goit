Я покажу как создать на github свой репозиторий и как работать с ним с локальной машины.
1. Регистрируемся на Github,
2. логинимся,
3. подтверждаем почту,
4. создаем новый репозиторий,
5. копируем ссылку на наш репозиторий,
6. в коммандной строке выполняем клонирование: git clone <url>,
7. переходим в локальный склонированый репозиторий,
8. git status - показывает состояние репозитория,
9. сохраним в репозиторий наш файл-инструкцию,
10. видим красным шрифтом название нашего файла - он еще не доавлен в репозиторий,
11. добавляем наш файл в локальный репозиторий: git add Github_instruction.txt
12. видим что файл стал зеленым - это значит что он уже есть в списке на добавление в репу,
13. теперь нам нужно зафиксировать файл в локальном репозитории: git commit -m "сообщение к комиту",
14. мы можем видеть что на github нашего файл еще нет (он есть только в локальном репозитории пока), 
чтобы файл появился на github, нужно вызвать комманду git push.
